# Shared Java Connectors – Known Issues & Troubleshooting

## 1. Known Issues

- External system outages or API changes can break connectors.

## 2. Troubleshooting

- When connector issues appear:
  - Check external system status and logs.
  - Inspect connector configuration and recent changes.

This document is self-contained and lists common issues and mitigations for the Shared Java Connectors library.
