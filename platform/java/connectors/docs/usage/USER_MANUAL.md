# Shared Java Connectors – User Manual

## 1. Audience

This manual is for backend engineers using `libs/java/connectors`.

## 2. Basic Usage

1. Add the library as a dependency in your service.
2. Use provided connector interfaces and helpers to integrate external systems.
3. Configure endpoints and credentials via shared configuration patterns.

This manual is self-contained and explains how to use the Shared Java Connectors library in typical workflows.
