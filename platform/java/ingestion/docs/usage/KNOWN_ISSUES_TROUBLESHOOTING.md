# Shared Java Ingestion – Known Issues & Troubleshooting

## 1. Known Issues

- Misconfigured batching or retries can overload downstream systems or drop work.

## 2. Troubleshooting

- When ingestion issues appear:
  - Check metrics for backlog, retries, and failures.
  - Inspect recent changes to ingestion configuration and pipeline definitions.

This document is self-contained and lists common issues and mitigations for the Shared Java Ingestion library.
