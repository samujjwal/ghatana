# Shared Java Ingestion – User Manual

## 1. Audience

This manual is for backend engineers using `libs/java/ingestion`.

## 2. Basic Usage

1. Add the library as a dependency in your service.
2. Use its helpers to build ingestion pipelines with appropriate batching and retry behavior.
3. Integrate metrics and logs with shared observability tooling.

This manual is self-contained and explains how to use the Shared Java Ingestion library in typical workflows.
