# @ghatana/charts

Shared chart primitives built on Recharts with Ghatana theming.

## Purpose

- expose reusable chart components and helpers
- align data visualization with shared theme and design tokens

## Dependencies

- peers on `react` and `recharts`
- depends on `@ghatana/theme` for visual consistency

## Boundaries

- no product-specific dashboards
- no application routing or page logic
