# Shared TypeScript Charts – Known Issues & Troubleshooting

## 1. Known Issues

- Chart rendering issues can arise from malformed data or breaking changes in underlying chart libs.

## 2. Troubleshooting

- When chart issues appear:
  - Inspect the data passed to chart components.
  - Check library versions and recent breaking changes.

This document is self-contained and lists common issues and mitigations for the Shared TypeScript Charts library.
