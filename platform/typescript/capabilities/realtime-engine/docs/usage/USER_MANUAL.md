# Shared TypeScript Realtime – User Manual

## 1. Audience

This manual is for frontend engineers using `libs/typescript/realtime`.

## 2. Basic Usage

1. Add the library as a dependency in your frontend app.
2. Use its hooks/utilities to connect to realtime backends and manage subscriptions.
3. Integrate received events with app state and UI components.

This manual is self-contained and explains how to use the Shared TypeScript Realtime library in typical workflows.
