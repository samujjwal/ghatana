# Shared TypeScript Realtime – Known Issues & Troubleshooting

## 1. Known Issues

- Network instability can cause frequent reconnects or dropped messages.
- Backend protocol or schema changes may break existing realtime flows.

## 2. Troubleshooting

- When realtime issues appear:
  - Check browser console and network tab for connection errors.
  - Verify backend realtime endpoints and protocols.
  - Inspect app logs for subscription or handler errors.

This document is self-contained and lists common issues and mitigations for the Shared TypeScript Realtime library.
