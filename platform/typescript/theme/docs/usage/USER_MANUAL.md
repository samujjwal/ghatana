# Shared TypeScript Theme – User Manual

## 1. Audience

This manual is for frontend engineers using `libs/typescript/theme`.

## 2. Basic Usage

1. Add the library as a dependency in your frontend app.
2. Import theme objects built on shared tokens.
3. Wrap your app in the provided theme context/provider and use hooks/utilities to access theme values.

This manual is self-contained and explains how to use the Shared TypeScript Theme library in typical workflows.
