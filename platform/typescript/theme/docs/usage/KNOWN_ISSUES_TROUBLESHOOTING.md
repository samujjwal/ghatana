# Shared TypeScript Theme – Known Issues & Troubleshooting

## 1. Known Issues

- Theme or token changes can cause visual regressions across multiple apps.

## 2. Troubleshooting

- When visual issues appear:
  - Check recent changes to `libs/typescript/theme` and `libs/typescript/tokens`.
  - Verify that apps have upgraded both libraries consistently.

This document is self-contained and lists common issues and mitigations for the Shared TypeScript Theme library.
