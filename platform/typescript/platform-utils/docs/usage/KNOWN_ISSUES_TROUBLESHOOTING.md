# Shared TypeScript Utils – Known Issues & Troubleshooting

## 1. Known Issues

- Changes to widely used utilities can cause regressions across many apps.

## 2. Troubleshooting

- When regressions appear:
  - Inspect recent changes in `libs/typescript/utils`.
  - Check how consuming projects rely on specific utility behavior.

This document is self-contained and lists common issues and mitigations for the Shared TypeScript Utils library.
