# Shared TypeScript Accessibility Audit – User Manual

## 1. Audience

This manual is for frontend engineers and QA engineers using `libs/typescript/accessibility-audit`.

## 2. Basic Usage

1. Add the library as a dependency in your project.
2. Use provided helpers in tests or dev tools to run accessibility audits.

This manual is self-contained and explains how to use the Shared TypeScript Accessibility Audit library in typical workflows.
