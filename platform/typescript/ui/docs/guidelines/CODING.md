# Shared TypeScript UI – Coding Guidelines

## 1. Scope

These guidelines apply to `libs/typescript/ui`.

## 2. Design Principles

- Use function components and hooks.
- Keep components presentational; delegate data fetching and mutations to hooks/services.
- Accept typed props aligned with shared contracts and domain models.

This document is self-contained and defines how to structure code in the Shared TypeScript UI library.
