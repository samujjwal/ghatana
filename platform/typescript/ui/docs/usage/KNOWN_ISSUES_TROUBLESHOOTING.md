# Shared TypeScript UI – Known Issues & Troubleshooting

## 1. Known Issues

- Changes to shared components can cause regressions across multiple apps.

## 2. Troubleshooting

- When UI regressions appear:
  - Check recent changes in `libs/typescript/ui`.
  - Inspect how apps compose and style the shared components.

This document is self-contained and lists common issues and mitigations for the Shared TypeScript UI library.
