# Shared TypeScript Tokens – Coding Guidelines

## 1. Scope

These guidelines apply to `libs/typescript/tokens`.

## 2. Design Principles

- Treat tokens as the single source of truth for design decisions.
- Keep tokens declarative (no logic) and framework-agnostic.
- Use clear naming conventions for colors, spacing, typography, etc.

This document is self-contained and defines how to structure code in the Shared TypeScript Tokens library.
