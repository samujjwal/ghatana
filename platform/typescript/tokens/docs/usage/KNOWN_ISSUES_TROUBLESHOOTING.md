# Shared TypeScript Tokens – Known Issues & Troubleshooting

## 1. Known Issues

- Token changes can cause visual regressions across multiple apps.

## 2. Troubleshooting

- When visual issues appear:
  - Check recent changes to tokens and their usage in UI/theme libraries.
  - Verify that apps have been updated to the correct token version.

This document is self-contained and lists common issues and mitigations for the Shared TypeScript Tokens library.
