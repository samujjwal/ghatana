# Shared TypeScript Tokens – User Manual

## 1. Audience

This manual is for frontend engineers using `libs/typescript/tokens`.

## 2. Basic Usage

1. Add the library as a dependency in your frontend app or UI/theme library.
2. Import and use token objects for styling (colors, spacing, typography, etc.).
3. Avoid redefining hard-coded values in apps; rely on tokens instead.

This manual is self-contained and explains how to use the Shared TypeScript Tokens library in typical workflows.
