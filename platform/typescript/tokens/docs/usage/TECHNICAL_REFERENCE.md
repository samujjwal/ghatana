# Shared TypeScript Tokens – Technical Reference

## 1. Overview

This reference summarizes key technical aspects of `libs/typescript/tokens`.

## 2. Core Concepts (Conceptual)

- Design tokens for colors, spacing, typography, radii, and other design primitives.
- Token objects consumed by UI and theme libraries.

This technical reference is self-contained and describes the primary surfaces of the Shared TypeScript Tokens library.
