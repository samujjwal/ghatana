# Shared TypeScript API – User Manual

## 1. Audience

This manual is for frontend engineers using `libs/typescript/api`.

## 2. Basic Usage

1. Add the library as a dependency in your app.
2. Use provided helpers and types to perform HTTP/GraphQL API calls.
3. Integrate API results with state and UI layers.

This manual is self-contained and explains how to use the Shared TypeScript API library in typical workflows.
