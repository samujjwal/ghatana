# Guardian – Backend API – Technical Reference

## 1. Overview

This reference summarizes key technical aspects of `@guardian/backend`.

## 2. Core Concepts (Conceptual)

- Fastify-based HTTP API with custom authentication.
- Postgres + Redis for persistence and caching.
- OpenTelemetry + Prometheus for observability.

## 3. Intended Consumers

- Guardian dashboard/mobile apps and internal services calling the backend API.

This technical reference is self-contained and describes the primary surfaces of the Guardian Backend API.
