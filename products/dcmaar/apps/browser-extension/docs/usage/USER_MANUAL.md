# Guardian – Browser Extension – User Manual

## 1. Audience

This manual is for operators and users deploying or using the Guardian browser extension.

## 2. Basic Usage

1. Install the extension into supported browsers.
2. Ensure configuration is in place for backend endpoints and policies.
3. Verify that performance data appears in Guardian dashboards.

This manual is self-contained and explains how to use the Guardian Browser Extension in typical workflows.
