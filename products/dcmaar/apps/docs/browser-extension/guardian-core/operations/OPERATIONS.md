# Guardian – Browser Extension – Operations Guide

## 1. Overview

`@yappc/guardian-browser-extension` is shipped as a browser extension for Chrome, Firefox, and Edge.

## 2. Build & Run

- Use `dev` for local development with Vite.
- Use `build` to produce browser-specific bundles via `build:chrome`, `build:firefox`, `build:edge`.

## 3. Monitoring

- Monitor extension errors via browser devtools and, where configured, remote logging.

This guide is self-contained and documents operational considerations for the Guardian Browser Extension.
