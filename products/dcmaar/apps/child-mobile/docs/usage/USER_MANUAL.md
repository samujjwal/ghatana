# Guardian – Parent Mobile – User Manual

## 1. Audience

This manual is for end users and operators using the Guardian Parent Mobile app.

## 2. Basic Usage

1. Install the app from the appropriate app store.
2. Sign in with Guardian credentials.
3. View activity and alerts, and adjust settings and policies as needed.

This manual is self-contained and explains how to use `@guardian/parent-mobile` in typical workflows.
