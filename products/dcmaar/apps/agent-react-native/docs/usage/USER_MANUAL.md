# Guardian – Agent React Native – User Manual

## 1. Audience

This manual is for engineers and operators working with `@guardian/agent-react-native`.

## 2. Basic Usage

1. Run the app with Expo during development.
2. Configure backend endpoints and credentials using environment/config.
3. Verify that data reaches Guardian backend and dashboards.

This manual is self-contained and explains how to use `@guardian/agent-react-native` in typical workflows.
