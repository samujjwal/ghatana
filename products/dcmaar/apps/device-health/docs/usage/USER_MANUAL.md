# DCMaar – Device Health Extension – User Manual

## 1. Audience

This manual is for end users and operators using the Device Health extension.

## 2. Basic Usage

1. Install the extension in supported browsers.
2. Connect it to the DCMaar backend/agent as required.
3. View device health dashboards and alerts in the extension UI.
4. Adjust settings (thresholds, notifications) via provided menus.

This manual is self-contained and explains how to use the Device Health extension in typical workflows.
