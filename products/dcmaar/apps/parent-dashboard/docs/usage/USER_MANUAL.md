# Guardian – Parent Dashboard – User Manual

## 1. Audience

This manual is for operators, product stakeholders, and engineers using the Guardian Parent Dashboard.

## 2. Basic Usage

1. Deploy the dashboard alongside the Guardian backend.
2. Access the dashboard in a browser and authenticate as a parent/administrator.
3. Navigate dashboards, alerts, and configuration pages to manage child profiles and policies.

This manual is self-contained and explains how to use the Guardian Parent Dashboard in typical workflows.
