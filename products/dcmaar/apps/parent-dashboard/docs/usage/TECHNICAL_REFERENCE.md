# Guardian – Parent Dashboard – Technical Reference

## 1. Overview

This reference summarizes key technical aspects of `@guardian/parent-dashboard`.

## 2. Core Concepts (Conceptual)

- React+Vite SPA using TailwindCSS and Jotai.
- Communication with Guardian backend via Axios and WebSocket (`socket.io-client`).

## 3. Intended Consumers

- Engineers extending or maintaining Guardian dashboard features.

This technical reference is self-contained and describes the primary surfaces of the Guardian Parent Dashboard.
