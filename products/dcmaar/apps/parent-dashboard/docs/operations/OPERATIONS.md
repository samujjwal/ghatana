# Guardian – Parent Dashboard – Operations Guide

## 1. Overview

`@guardian/parent-dashboard` is a web app. Operations include building, running, and deploying the dashboard.

## 2. Build & Run

- Use `dev` to start Vite dev server via `scripts/start-vite.cjs`.
- Use `build` and `preview` for production builds and local previews.
- Docker support via `docker:build`, `docker:run`, `docker:dev`, `docker:prod`, `docker:stop`.

## 3. Monitoring

- Integrate with Sentry and web-vitals for client-side monitoring.
- Monitor backend connectivity and error rates via logs and metrics.

This guide is self-contained and documents operational considerations for the Guardian Parent Dashboard.
