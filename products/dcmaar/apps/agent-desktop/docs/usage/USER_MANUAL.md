# Guardian – Agent Desktop – User Manual

## 1. Audience

This manual is for operators and engineers enabling desktop collection for Guardian.

## 2. Basic Usage

1. Ensure the DCMAAR agent daemon includes `guardian-agent-desktop` in its plugin set.
2. Configure which signals to collect and any OS-specific filters.
3. Validate data flow from desktop devices into Guardian dashboards.

This manual is self-contained and explains how to use the Guardian Agent Desktop plugin in typical workflows.
