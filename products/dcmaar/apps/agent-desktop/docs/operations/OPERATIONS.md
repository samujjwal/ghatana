# Guardian – Agent Desktop – Operations Guide

## 1. Overview

`guardian-agent-desktop` is a plugin loaded by the DCMAAR agent daemon. Operations focus on rollout, observability, and incident handling.

## 2. Deployment & Configuration

- Ship the plugin as part of the agent daemon plugin set.
- Enable or disable collection via agent configuration and capability flags.

## 3. Monitoring

- Monitor plugin logs via `tracing` output from the agent.
- Track event throughput and error rates through agent metrics.

This guide is self-contained and documents operational considerations for the Guardian Agent Desktop plugin.
