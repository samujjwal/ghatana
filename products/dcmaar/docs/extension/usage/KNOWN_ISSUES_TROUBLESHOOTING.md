# DCMaar Browser Extension – Known Issues & Troubleshooting

## 1. Known Issues

- **Store distribution delays** can slow down rollout.
- **Browser-specific quirks** may affect behavior on some sites.

## 2. Troubleshooting Scenarios

### 2.1 Extension Not Capturing as Expected

- **Checks**:
  - Verify the site is on the allowlist.
  - Confirm extension is enabled and has necessary permissions.

### 2.2 Performance Issues on Certain Sites

- **Checks**:
  - Inspect content scripts and event listeners.
  - Review configuration to minimize heavy capture on complex pages.

This document is self-contained and lists common issues and mitigations for the DCMaar Browser Extension.
