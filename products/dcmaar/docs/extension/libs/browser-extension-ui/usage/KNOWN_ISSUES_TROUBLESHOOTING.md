# DCMaar – Browser Extension UI – Known Issues & Troubleshooting

## 1. Known Issues

- Component changes can cause layout or styling regressions across extension views.

## 2. Troubleshooting

- When regressions appear, check:
  - Recent changes in shared components or styles.

This document is self-contained and lists common issues and mitigations for `@dcmaar/browser-extension-ui`.
