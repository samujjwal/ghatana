# DCMaar – Plugin Abstractions – Known Issues & Troubleshooting

## 1. Known Issues

- Interface changes can break plugins or hosts if not coordinated.

## 2. Troubleshooting

- When plugins fail to load or behave unexpectedly, check:
  - Package and interface versions.
  - Changes in required methods or fields.

This document is self-contained and lists common issues and mitigations for `@dcmaar/plugin-abstractions`.
