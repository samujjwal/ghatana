# DCMaar – Plugin Abstractions – User Manual

## 1. Audience

This manual is for TypeScript engineers implementing plugins or hosts using `@dcmaar/plugin-abstractions`.

## 2. Basic Usage

1. Add `@dcmaar/plugin-abstractions` and `@dcmaar/types` as dependencies.
2. Implement the provided plugin interfaces in plugin code.
3. Use reference implementations as examples when integrating into hosts.

This manual is self-contained and explains how to use `@dcmaar/plugin-abstractions` in typical flows.
