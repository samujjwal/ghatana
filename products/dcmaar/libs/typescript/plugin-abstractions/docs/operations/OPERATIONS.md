# DCMaar – Plugin Abstractions – Operations Guide

## 1. Overview

`@dcmaar/plugin-abstractions` is a library; operational impact arises from how plugin hosts and plugins adopt new versions.

## 2. Versioning

- Coordinate updates with plugin hosts and plugin authors.
- Avoid breaking changes without a clear migration path.

This guide is self-contained and documents operational considerations for `@dcmaar/plugin-abstractions`.
