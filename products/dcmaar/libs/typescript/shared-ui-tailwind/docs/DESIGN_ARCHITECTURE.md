# DCMaar – Shared UI Tailwind – Design & Architecture

## 1. Purpose

`@dcmaar/shared-ui-tailwind` provides **shared Tailwind CSS components** (buttons, cards, inputs, etc.) for DCMaar React apps.

## 2. Responsibilities

- Define reusable Tailwind-based components aligned with DCMaar design tokens.

## 3. Architectural Position

- React component library, depends on `@dcmaar/shared-ui-core` for tokens/types.

This document is self-contained and summarizes the architecture and role of `@dcmaar/shared-ui-tailwind`.
