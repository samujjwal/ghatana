# DCMaar – Shared UI Tailwind – Coding Guidelines

## 1. Scope

These guidelines apply to `@dcmaar/shared-ui-tailwind`.

## 2. Component Design

- Keep Tailwind-based components small and composable.
- Use tokens and utilities from `@dcmaar/shared-ui-core` where possible.

## 3. Dependencies

- React/ReactDOM and TailwindCSS as peers; do not embed app-specific state.

This document is self-contained and defines how to structure code in `@dcmaar/shared-ui-tailwind`.
