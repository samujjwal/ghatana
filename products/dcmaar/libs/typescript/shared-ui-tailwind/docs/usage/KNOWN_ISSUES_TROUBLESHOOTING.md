# DCMaar – Shared UI Tailwind – Known Issues & Troubleshooting

## 1. Known Issues

- Style or component changes can cause regressions in many apps simultaneously.

## 2. Troubleshooting

- When regressions appear, inspect recent changes to `@dcmaar/shared-ui-tailwind` and how apps consume its components.

This document is self-contained and lists common issues and mitigations for `@dcmaar/shared-ui-tailwind`.
