# DCMaar – Shared UI Tailwind – User Manual

## 1. Audience

This manual is for React engineers using `@dcmaar/shared-ui-tailwind`.

## 2. Basic Usage

1. Add `@dcmaar/shared-ui-tailwind` and `@dcmaar/shared-ui-core` as dependencies.
2. Import components from this package into app UIs.

This manual is self-contained and explains how to use `@dcmaar/shared-ui-tailwind` in typical flows.
