# DCMaar – Types – Known Issues & Troubleshooting

## 1. Known Issues

- Breaking changes in shared types can affect many packages at once.

## 2. Troubleshooting

- When compilation errors appear across multiple packages, check:
  - `@dcmaar/types` version changes.
  - Recent modifications to exported types.

This document is self-contained and lists common issues and mitigations for `@dcmaar/types`.
