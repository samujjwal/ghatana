# DCMaar – Types – User Manual

## 1. Audience

This manual is for TypeScript engineers using `@dcmaar/types` in other packages.

## 2. Basic Usage

1. Add `@dcmaar/types` as a dependency.
2. Import and reuse shared types from the root or from `./common`, `./errors`, and `./utils`.

## 3. Best Practices

- Prefer using existing types from this package over redefining equivalents.

This manual is self-contained and explains how to use `@dcmaar/types` in typical flows.
