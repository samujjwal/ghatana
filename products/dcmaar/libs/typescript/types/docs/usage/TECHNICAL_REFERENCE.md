# DCMaar – Types – Technical Reference

## 1. Overview

This reference summarizes key technical concepts of `@dcmaar/types`.

## 2. Core Concepts (Conceptual)

- Core domain models shared across DCMAAR packages.
- Common error types and utility types exposed via `./common`, `./errors`, and `./utils`.

## 3. Intended Consumers

- All TypeScript packages in the DCMaar ecosystem that need shared types.

This technical reference is self-contained and describes the primary surfaces of `@dcmaar/types`.
