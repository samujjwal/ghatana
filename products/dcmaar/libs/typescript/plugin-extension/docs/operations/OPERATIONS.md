# DCMaar – Plugin Extension – Operations Guide

## 1. Overview

`@dcmaar/plugin-extension` is a library; operational impact comes from how its plugins are used in apps such as `device-health`.

## 2. Configuration

- Configure plugin sampling intervals, thresholds, and enabled/disabled state via the host app configuration.

## 3. Monitoring

- Monitor plugin-produced metrics and events through the app’s telemetry and dashboards.
- Watch for anomalies such as missing data or sudden jumps in metrics.

This guide is self-contained and documents operational considerations for `@dcmaar/plugin-extension`.
