# DCMaar – Plugin Extension – Technical Reference

## 1. Overview

This reference summarizes key technical concepts of `@dcmaar/plugin-extension`.

## 2. Core Concepts (Conceptual)

- Concrete device monitoring plugins implemented using `@dcmaar/plugin-abstractions` and `@dcmaar/types`.
- Plugin outputs that integrate with host apps (e.g., `device-health`) via shared contracts.

## 3. Intended Consumers

- Apps that embed device monitoring plugins, and engineers extending the plugin set.

This technical reference is self-contained and describes the primary surfaces of `@dcmaar/plugin-extension`.
