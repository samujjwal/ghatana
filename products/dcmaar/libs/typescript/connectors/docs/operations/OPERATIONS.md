# DCMaar – Connectors – Operations Guide

## 1. Overview

`@dcmaar/connectors` is a library; operational impact is via how it is configured and used in apps.

## 2. Configuration

- Configure endpoints, timeouts, retries, and batch sizes via consuming app configs.

## 3. Monitoring

- Monitor connector metrics: error rates, latency, queue sizes.

This guide is self-contained and documents operational considerations for `@dcmaar/connectors`.
