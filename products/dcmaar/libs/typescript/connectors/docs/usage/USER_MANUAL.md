# DCMaar – Connectors – User Manual

## 1. Audience

This manual is for engineers using `@dcmaar/connectors` in DCMaar apps.

## 2. Basic Usage

1. Add `@dcmaar/connectors` as a dependency.
2. Configure connectors with appropriate endpoints and resilience settings.

This manual is self-contained and explains how to use `@dcmaar/connectors` in typical flows.
