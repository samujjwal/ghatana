# DCMaar – Agent UI – Known Issues & Troubleshooting

## 1. Known Issues

- Changes to shared components can cause visual regressions in multiple apps.

## 2. Troubleshooting

- When regressions appear, inspect recent `@dcmaar/agent-ui` changes and app usage of affected components.

This document is self-contained and lists common issues and mitigations for `@dcmaar/agent-ui`.
