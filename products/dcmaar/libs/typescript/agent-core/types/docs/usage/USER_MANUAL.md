# DCMaar – Agent Types – User Manual

## 1. Audience

This manual is for TypeScript engineers who need canonical agent domain types.

## 2. Basic Usage

1. Add `@dcmaar/agent-types` as a dependency.
2. Import the relevant types and use them in function signatures, stores, and API clients.

This manual is self-contained and explains how to use `@dcmaar/agent-types` in typical workflows.
