# DCMaar – Agent Types – Known Issues & Troubleshooting

## 1. Known Issues

- Type changes made without coordinating with runtimes can cause runtime/type mismatches.

## 2. Troubleshooting

- When mismatches appear, inspect:
  - Recent changes in `@dcmaar/agent-types`.
  - Generated or hand-written clients and their expectations.

This document is self-contained and lists common issues and mitigations for `@dcmaar/agent-types`.
