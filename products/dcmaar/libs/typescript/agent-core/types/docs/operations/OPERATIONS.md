# DCMaar – Agent Types – Operations Guide

## 1. Overview

`@dcmaar/agent-types` is a pure types package; operational impact is via the callers that depend on it.

## 2. Versioning

- Coordinate type changes with Rust agents, bridge-protocol, and Guardian apps.
- Use semantic versioning and treat type removals or incompatible changes as breaking.

This guide is self-contained and documents operational considerations for `@dcmaar/agent-types`.
