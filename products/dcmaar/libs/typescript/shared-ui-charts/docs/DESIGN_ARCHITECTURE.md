# DCMaar – Shared UI Charts – Design & Architecture

## 1. Purpose

`@dcmaar/shared-ui-charts` provides **shared chart components** for DCMaar UIs, built on Recharts and shared tailwind UI primitives.

## 2. Responsibilities

- Offer reusable chart components wired to DCMaar design tokens and tailwind styles.
- Keep chart logic generic and data-agnostic.

## 3. Architectural Position

- React-based chart library, depends on `@dcmaar/shared-ui-tailwind` and `recharts`.

This document is self-contained and summarizes the architecture and role of `@dcmaar/shared-ui-charts`.
