# DCMaar – Shared UI Charts – User Manual

## 1. Audience

This manual is for React engineers using `@dcmaar/shared-ui-charts` in dashboards and apps.

## 2. Basic Usage

1. Add `@dcmaar/shared-ui-charts` and its peers as dependencies.
2. Import chart components and bind them to domain data.

This manual is self-contained and explains how to use `@dcmaar/shared-ui-charts` in typical flows.
