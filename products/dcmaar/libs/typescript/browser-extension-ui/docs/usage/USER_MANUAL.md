# DCMaar – Browser Extension UI – User Manual

## 1. Audience

This manual is for React engineers building DCMaar browser extensions.

## 2. Basic Usage

1. Add `@dcmaar/browser-extension-ui` and `@dcmaar/browser-extension-core` as dependencies.
2. Compose the provided components into extension UIs.

This manual is self-contained and explains how to use `@dcmaar/browser-extension-ui` in typical flows.
