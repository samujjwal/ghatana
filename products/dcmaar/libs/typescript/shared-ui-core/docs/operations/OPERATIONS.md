# DCMaar – Shared UI Core – Operations Guide

## 1. Overview

`@dcmaar/shared-ui-core` is a library; operational impact is via its adoption by multiple UI packages.

## 2. Versioning

- Coordinate token and hook changes with dependent UIs (desktop, web, extension UIs).
- Use semver to signal breaking style or API changes.

This guide is self-contained and documents operational considerations for `@dcmaar/shared-ui-core`.
