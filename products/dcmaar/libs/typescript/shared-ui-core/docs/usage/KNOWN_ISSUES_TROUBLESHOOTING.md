# DCMaar – Shared UI Core – Known Issues & Troubleshooting

## 1. Known Issues

- Token or hook changes can cause visual or behavioral regressions across multiple UIs.

## 2. Troubleshooting

- When UI regressions appear, check:
  - Recent changes in `@dcmaar/shared-ui-core`.
  - Token and hook usage patterns in affected components.

This document is self-contained and lists common issues and mitigations for `@dcmaar/shared-ui-core`.
