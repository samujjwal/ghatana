# DCMaar – Config Presets – Known Issues & Troubleshooting

## 1. Known Issues

- Preset changes can cause subtle behavior changes across multiple apps.

## 2. Troubleshooting

- When issues arise, check:
  - Which preset version each component is using.
  - Schema validation results for customized presets.

This document is self-contained and lists common issues and mitigations for `@dcmaar/config-presets`.
