# DCMaar – Config Presets – User Manual

## 1. Audience

This manual is for engineers configuring DCMaar components using `@dcmaar/config-presets`.

## 2. Basic Usage

1. Add `@dcmaar/config-presets` as a dependency.
2. Import presets and, if necessary, refine them for a specific environment.

This manual is self-contained and explains how to use `@dcmaar/config-presets` in typical flows.
