# DCMaar – Config Presets – Operations Guide

## 1. Overview

`@dcmaar/config-presets` is a library; operational impact is via how presets are used in agents, desktop, and extensions.

## 2. Versioning & Rollout

- Coordinate preset changes with consuming services and apps.
- Roll out new presets gradually in staging before production.

This guide is self-contained and documents operational considerations for `@dcmaar/config-presets`.
