# DCMaar – Config Presets – Design & Architecture

## 1. Purpose

`@dcmaar/config-presets` provides **shared configuration presets** for the DCMaar platform (agent, desktop, extension).

## 2. Responsibilities

- Define reusable, validated configuration bundles.
- Provide Zod schemas for preset validation.

## 3. Architectural Position

- Pure TS library used by multiple apps and services for consistent configuration.

This document is self-contained and summarizes the architecture and role of `@dcmaar/config-presets`.
