# DCMaar Common – User Manual

## 1. Audience

This manual is for Rust engineers using `dcmaar-common` in other crates.

## 2. Basic Usage

1. Add `dcmaar-common` as a dependency via the workspace.
2. Use its error types and helpers instead of redefining equivalents.

This manual is self-contained and explains how to use `dcmaar-common` in typical flows.
