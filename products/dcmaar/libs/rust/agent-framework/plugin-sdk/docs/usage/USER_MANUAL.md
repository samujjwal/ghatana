# DCMaar Plugin SDK – User Manual

## 1. Audience

This manual is for plugin authors building against the DCMaar agent framework.

## 2. Basic Usage

1. Add `dcmaar-plugin-sdk` as a dependency in the plugin project.
2. Implement the provided traits and use SDK types.
3. Build and package plugins according to platform guidelines.

This manual is self-contained and explains how to use `dcmaar-plugin-sdk` in typical flows.
