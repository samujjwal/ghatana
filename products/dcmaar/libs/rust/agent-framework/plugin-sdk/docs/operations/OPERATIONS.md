# DCMaar Plugin SDK – Operations Guide

## 1. Overview

`dcmaar-plugin-sdk` is a library for plugin authors. Operational concerns relate to how SDK changes roll out to plugins.

## 2. Versioning & Distribution

- Coordinate SDK version bumps with plugin builds.
- Provide clear changelogs for plugin authors.

This guide is self-contained and documents operational considerations for `dcmaar-plugin-sdk`.
