# DCMaar Agent Common – Technical Reference

## 1. Overview

This reference summarizes the technical role of `dcmaar-agent-common`.

## 2. Core Concepts (Conceptual)

- Shared agent domain models and configuration types.
- Validation and helper functions for agent runtimes.
- Optional integrations for storage, gRPC, and WASM features.

## 3. Intended Consumers

- DCMaar agent daemon and other Rust-based agent components.

This technical reference is self-contained and describes the primary surfaces of `dcmaar-agent-common`.
