# DCMaar Browser Extension – User Manual

## 1. Audience

This manual is for end users and operators using the DCMaar Browser Extension.

## 2. Basic Usage

1. Install the extension from the approved store or internal distribution.
2. Sign in or connect it to the DCMaar platform if required.
3. Confirm that it reports a healthy connection and is active on allowed domains.

## 3. Best Practices

- Keep the extension updated to the latest approved version.
- Review privacy and capture settings regularly.

This manual is self-contained and explains how to use the DCMaar Browser Extension in typical flows.
