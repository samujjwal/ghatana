# DCMaar Server – Technical Reference

## 1. Overview

This reference summarizes the main technical surfaces of the DCMaar Server.

## 2. Core Concepts (Conceptual)

- Ingest pipelines for agent/extension events.
- Policy enforcement using OPA.
- Storage and query layers for structured data.
- Observability endpoints for metrics and traces.

## 3. Intended Consumers

- Desktop app and other DCMaar clients.
- Operators and platform engineers integrating new data sources and queries.

This technical reference is self-contained and describes the primary surfaces of the DCMaar Server.
