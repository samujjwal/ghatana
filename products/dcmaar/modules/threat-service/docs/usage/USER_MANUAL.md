# DCMaar Server – User Manual

## 1. Audience

This manual is for operators and platform engineers responsible for running the DCMaar Server.

## 2. Basic Usage

1. Deploy server with configured DB, Redis, and OPA bundles.
2. Point agents, desktop, and extensions at the server endpoints.
3. Verify ingest and query flows using health and status endpoints.
4. Monitor metrics and logs to ensure healthy operation.

## 3. Best Practices

- Keep separate configs for dev, staging, and production.
- Use IaC and CI/CD to manage deployments and schema migrations.

This manual is self-contained and explains how to deploy and operate the DCMaar Server.
