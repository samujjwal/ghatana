# DCMaar Desktop – User Manual

## 1. Audience

This manual is for end users and operators using the DCMaar Desktop application.

## 2. Basic Usage

1. Install the desktop application.
2. Connect to the DCMaar Server and sign in if required.
3. Configure connectors and view dashboards.
4. Use built-in workflows to inspect and manage agent/extension behavior.

## 3. Best Practices

- Start in a staging environment before connecting to production.
- Use least-privilege accounts when interacting with sensitive data.

This manual is self-contained and explains how to use the DCMaar Desktop app in typical workflows.
