# DCMaar Desktop – Technical Reference

## 1. Overview

This reference summarizes technical surfaces of the DCMaar Desktop application.

## 2. Core Concepts (Conceptual)

- Desktop shell combining React UI with Tauri backend.
- Data flows from server and agent into UI dashboards and configuration screens.
- Connector configuration and management flows.

## 3. Intended Consumers

- Engineers extending the desktop app UI or Tauri backend.

This technical reference is self-contained and describes the primary surfaces of the DCMaar Desktop app.
