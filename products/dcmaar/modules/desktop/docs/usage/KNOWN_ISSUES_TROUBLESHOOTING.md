# DCMaar Desktop – Known Issues & Troubleshooting

## 1. Known Issues

- **Connectivity problems** between desktop and server/agent.
- **Rendering or performance issues** on constrained machines.

## 2. Troubleshooting Scenarios

### 2.1 Cannot Connect to Server

- **Checks**:
  - Validate endpoint configuration.
  - Check network and proxy settings.

### 2.2 UI Feels Slow

- **Checks**:
  - Inspect large dashboards or heavy queries.
  - Review logs for repeated errors.

This document is self-contained and lists common issues and mitigations for the DCMaar Desktop app.
