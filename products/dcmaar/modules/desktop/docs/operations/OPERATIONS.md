# DCMaar Desktop – Operations Guide

## 1. Overview

The DCMaar Desktop app is a packaged desktop client. Operations focus on distribution, configuration, and monitoring.

## 2. Distribution & Updates

- Built and packaged via Tauri and frontend build pipeline.
- Distributed via standard desktop app channels (installers, stores, internal tools).
- Use Tauri update mechanisms to manage updates.

## 3. Configuration

- Configure server endpoints and local agent connectivity via UI and/or config files.
- Respect environment variables and OS-specific configuration where used.

## 4. Monitoring & Support

- Capture application logs and telemetry (with user consent and privacy controls).
- Provide clear error messages for connectivity and configuration issues.

This guide is self-contained and documents operational considerations for the DCMaar Desktop app.
