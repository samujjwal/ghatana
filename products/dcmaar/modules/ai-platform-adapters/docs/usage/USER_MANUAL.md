# DCMaar Agent Daemon – User Manual

## 1. Audience

This manual is for operators and platform engineers responsible for rolling out and managing the DCMaar Agent Daemon.

## 2. Basic Usage

1. Install the agent package on target machines.
2. Provide configuration (server endpoints, TLS, capture scopes) via config files or managed tooling.
3. Start the agent service and verify connectivity and telemetry.
4. Optionally enable plugins and advanced capture scenarios.

## 3. Best Practices

- Start with minimal capture and gradually expand.
- Use fleet tooling to manage configuration at scale.
- Regularly review metrics and logs for anomalies.

This manual is self-contained and explains how to deploy and operate the DCMaar Agent Daemon.
