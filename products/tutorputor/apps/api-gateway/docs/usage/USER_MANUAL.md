# Tutorputor API Gateway – User Manual

## 1. Audience

This manual is for engineers and operators using the Tutorputor API Gateway.

## 2. Basic Usage

1. Deploy the gateway alongside Tutorputor backend services.
2. Configure upstream service endpoints, auth, and rate-limiting.
3. Point Tutorputor web and mobile clients at the gateway’s base URL.

This manual is self-contained and explains how to use the Tutorputor API Gateway in typical workflows.
