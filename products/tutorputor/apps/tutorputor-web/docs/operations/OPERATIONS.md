# Tutorputor Web App – Operations Guide

## 1. Overview

The Tutorputor Web app is deployed as part of the frontend stack and relies on the Tutorputor API Gateway and backend services.

## 2. Deployment & Configuration

- Build using the standard frontend toolchain.
- Configure API Gateway base URLs, auth settings, and environment-specific flags at runtime.

## 3. Monitoring

- Track frontend errors and performance (e.g., via browser monitoring and logs).
- Watch request patterns and error codes from the API Gateway.

This guide is self-contained and documents operational considerations for the Tutorputor Web app.
