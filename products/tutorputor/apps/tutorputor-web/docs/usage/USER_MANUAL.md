# Tutorputor Web App – User Manual

## 1. Audience

This manual is for learners, instructors, admins, and engineers using the Tutorputor Web app.

## 2. Basic Usage

1. Sign in via the configured auth provider.
2. Navigate learning, assessment, analytics, and marketplace sections.
3. Use filters and controls in each view to perform day‑to‑day tasks.

This manual is self-contained and explains how to use the Tutorputor Web app in typical workflows.
