# Tutorputor Web App – Known Issues & Troubleshooting

## 1. Known Issues

- Backend/API failures or auth problems surface as UI errors or missing data.
- Long-running backend operations may lead to timeouts or confusing states if not handled.

## 2. Troubleshooting

- When users encounter issues:
  - Check browser console and network logs for failed requests.
  - Verify API Gateway health and configuration.
  - Confirm user permissions and feature flags.

This document is self-contained and lists common issues and mitigations for the Tutorputor Web app.
