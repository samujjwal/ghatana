# Tutorputor – User Manual

## 1. Audience

This manual is for engineers and operators working with Tutorputor.

## 2. Basic Usage

1. Define or update tutoring contracts in `contracts/`.
2. Implement or extend flows in `services/`.
3. Wire `apps/` to call services and present the experience.

This manual is self-contained and explains how to use Tutorputor in typical workflows.
