# Content Phase 2 Proof Runbook

## Purpose
Operational runbook for TP-008 and TP-009 phase 2 content-route proof execution.

## Preconditions
- TutorPutor platform dependencies installed.
- Environment credentials and test data ready.
- Operator has access for local, staging, preprod, and production capture.

## Steps
1. Run phase 2 route verification script.
2. Capture local command and response evidence.
3. Execute staging wrapper and capture outputs.
4. Execute preprod wrapper and capture outputs.
5. Execute production wrapper and capture outputs.
6. Update execution log and status matrix.
7. Complete signoff once evidence review passes.
