# LTI Phase 2 Proof Runbook

## Purpose
Runbook for LTI phase 2 boundary-validation verification and evidence collection.

## Preconditions
- TutorPutor platform dependencies installed.
- LTI test fixtures and environment vars available.
- Operator access for local/staging/preprod/production evidence updates.

## Steps
1. Execute local LTI phase 2 verification script.
2. Capture command output and annotate local evidence file.
3. Execute staging wrapper and record results.
4. Execute preprod wrapper and record results.
5. Execute production wrapper in approved change window.
6. Update status matrix and execution log.
7. Complete signoff document.
