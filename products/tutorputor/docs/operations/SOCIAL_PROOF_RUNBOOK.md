# Social Proof Runbook

## Objective

Capture repeatable evidence for social route validation and critical social flows.

## Steps

1. Run social validation tests.
2. Call social verification script against target environment.
3. Save output logs and correlation IDs.
4. Populate environment evidence file.
5. Update social status matrix and signoff file.

## Commands

- scripts/collect-social-validation-evidence.ps1
- scripts/collect-social-validation-evidence.sh
- scripts/verify-social-routes.ps1
- scripts/verify-social-routes.sh
