# Tutorputor AI Proxy Service – Known Issues & Troubleshooting

## 1. Known Issues

- Provider outages or API changes can surface as proxy errors.
- Misconfigured rate limits or timeouts can degrade Tutorputor experiences.

## 2. Troubleshooting

- When proxy calls fail:
  - Check provider status dashboards and logs.
  - Inspect proxy logs for error codes and safety filter triggers.
  - Verify credentials, endpoints, and configuration.

This document is self-contained and lists common issues and mitigations for the Tutorputor AI Proxy service.
