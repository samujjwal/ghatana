# Tutorputor DB Service – Known Issues & Troubleshooting

## 1. Known Issues

- Schema or migration drift can cause runtime errors across Tutorputor services.

## 2. Troubleshooting

- When DB-related issues arise:
  - Check migration history and schema differences across environments.
  - Inspect logs for constraint violations and connection issues.

This document is self-contained and lists common issues and mitigations for the Tutorputor DB service/module.
