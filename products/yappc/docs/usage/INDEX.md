# YAPPC – Usage, Reference & Backlog

This directory groups **usage-oriented** documentation for YAPPC.

## Planned documents

- `TECHNICAL_REFERENCE.md` – Key APIs, modules, and shared packages exposed by YAPPC.
- `USER_MANUAL.md` – How end users and teams use the YAPPC app-creator and related tools.
- `KNOWN_ISSUES_TROUBLESHOOTING.md` – Known issues and fix/workaround playbook.
- `FUTURE_BACKLOG.md` – YAPPC-specific backlog items not yet captured in the global roadmap.

These docs build on the implementation roadmap and will help consumers understand how to integrate with and operate YAPPC.
