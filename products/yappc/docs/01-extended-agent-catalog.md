# Canonical Extended Agent Catalog
Product lifecycle: Ideation → Planning → Design → Build → Test → Deploy → Operate → Enhance

This catalog is organized by domain.
Naming convention:
- Orchestrator: coordinates a domain
- CapabilityAgent: owns a reusable capability cluster
- TaskAgent: performs bounded workflow steps
- MicroAgent: atomic reusable execution unit
