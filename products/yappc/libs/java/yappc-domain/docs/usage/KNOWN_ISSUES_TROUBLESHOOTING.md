# YAPPC Domain – Known Issues & Troubleshooting

## 1. Known Issues

- Weak or underspecified invariants can lead to inconsistent YAPPC state.

## 2. Troubleshooting

- When domain issues appear:
  - Inspect domain types and invariants for gaps.
  - Check for callers constructing domain objects incorrectly.

This document is self-contained and lists common issues and mitigations for the YAPPC Domain library.
