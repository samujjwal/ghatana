# YAPPC Core – KG CLI Tools – Known Issues & Troubleshooting

## 1. Known Issues

- **Environment mismatches** – Commands may behave differently if configuration is inconsistent across environments.

## 2. Troubleshooting Scenarios

### 2.1 CLI Cannot Connect to Target

- **Checks**:
  - Validate endpoint URLs and ports.
  - Confirm network access and credentials.

### 2.2 Unexpected Output Format

- **Checks**:
  - Verify flags (e.g., JSON vs CSV output) and configuration.

This document is self-contained and lists common KG CLI Tools issues and mitigations.
