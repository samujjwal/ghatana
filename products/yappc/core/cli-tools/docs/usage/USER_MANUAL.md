# YAPPC Core – KG CLI Tools – User Manual

## 1. Audience

This manual is for engineers and operators who use the KG CLI Tools in daily work or automation.

## 2. Basic Workflow

1. Install or make the CLI available in your environment.
2. Configure connection parameters (endpoints, credentials) via config or env vars.
3. Use read-only commands to inspect data.
4. Use modification commands with care (ideally with dry-run options).

## 3. Best Practices

- Version control scripts that invoke the CLI.
- Log command usage in CI to aid audit and troubleshooting.

This manual is self-contained and explains how to use KG CLI Tools in typical workflows.
