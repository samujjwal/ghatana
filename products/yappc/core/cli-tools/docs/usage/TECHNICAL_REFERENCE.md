# YAPPC Core – KG CLI Tools – Technical Reference

## 1. Overview

This reference summarizes the key command-line surfaces provided by the KG CLI Tools module.

## 2. Core Concepts (Conceptual)

- **Commands** – Top-level operations (e.g., inspect, export, validate) executed from the shell.
- **Arguments/Flags** – Control behavior (targets, filters, formats).
- **Configuration** – Shared options for environment, auth, and output.

## 3. Intended Users

- Engineers needing a scriptable way to interact with the Knowledge Graph.
- Operators and CI jobs performing maintenance and checks.

This technical reference is self-contained and describes how to think about KG CLI Tools at a technical level.
