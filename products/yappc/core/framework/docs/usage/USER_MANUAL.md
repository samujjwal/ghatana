# YAPPC Core – Framework – User Manual

## 1. Audience

This manual is for engineers building or maintaining YAPPC services that consume the Core Framework.

## 2. Basic Usage

1. Add the framework module as a dependency.
2. Use the provided HTTP configuration and filters to set up controllers.
3. Integrate observability via framework helpers.

## 3. Best Practices

- Keep service-specific logic in the service, not in the framework.
- Reuse framework helpers rather than re-implementing infrastructure patterns.

This manual is self-contained and explains how to adopt the YAPPC Core Framework in services.
