# YAPPC Core – Scaffold – User Manual

## 1. Audience

This manual is for developers using Scaffold to generate new YAPPC modules.

## 2. Basic Usage

1. Choose the appropriate template pack (e.g., backend service, TS library, Virtual-Org integration).
2. Run the CLI command with required options (project name, package paths, etc.).
3. Review the generated structure and follow post-generation instructions (if any).

## 3. Best Practices

- Keep generated code under version control.
- Customize only where necessary; prefer regenerating or updating packs when standards evolve.

This manual is self-contained and explains how to use the Scaffold tooling.
