# Workspace Components

Components for workspace-centric navigation, state display, and user actions.

## Scope
- Shared workspace UI used across onboarding, management, and day-to-day flows.
- Components that expose workspace membership, project grouping, or context.
- Product-local workspace presentation logic.

## Key Areas
- Workspace context components.
- Shared management or selection widgets.
- Supporting workflow helpers tied to workspace state.
