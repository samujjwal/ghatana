# UI Components

Product-local UI components that complement the shared design system in the YAPPC web app.

## Scope
- Reusable presentational components.
- Product-specific wrappers and UI affordances.
- Shared visual building blocks used across routes and pages.

## Key Areas
- Reusable product-facing widgets.
- Lightweight wrappers around shared design-system primitives.
- Supporting local UI helpers.
