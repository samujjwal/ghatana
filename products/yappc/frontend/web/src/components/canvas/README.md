# Canvas Components

Primary canvas UI composition layer for YAPPC web editing, visualization, and interaction workflows.

## Scope
- Core canvas components and supporting submodules.
- Shared editing, layout, and visualization behavior used by canvas routes.
- Product-local canvas presentation separate from service and persistence layers.

## Key Areas
- Canvas shells, panels, nodes, tools, and unified submodules.
- Route-supporting editing and visualization components.
- Shared canvas-specific UI infrastructure.
