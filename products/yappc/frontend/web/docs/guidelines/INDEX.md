# YAPPC App-Creator Web – Guidelines

This directory groups **guideline** docs specific to the YAPPC app-creator web app.

## Planned documents

- `CODING.md` – React/TypeScript coding conventions for this app.
- `TESTING.md` – Testing strategy (unit, integration, E2E) and coverage goals.
- `QUALITY.md` – UX/performance expectations (e.g., canvas responsiveness).
- `SECURITY.md` – Frontend security aspects (auth integration, XSS, CSRF).
- `BUILD.md` – Build steps (Vite, test configs) and CI checks.
- `TOOLS_LIBS.md` – Libraries and tooling used by this app.
- `CHANGE_MANAGEMENT_DEPRECATION.md` – How to evolve app-creator features safely.

These will extend the YAPPC-wide guidelines with app-specific details.
