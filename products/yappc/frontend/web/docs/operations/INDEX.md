# YAPPC App-Creator Web – Operations

This directory groups **operations-focused** docs for the YAPPC app-creator web app.

## Planned documents

- `OPERATIONS.md` – Runtime expectations and operational considerations for the web app.
- `DEPLOYMENT_MONITORING.md` – Deployment flow, hosting environment, monitoring, alerts.
- `SETUP_GUIDE.md` – Local/dev setup (env vars, scripts, backend dependencies).
- `LIFECYCLE_DEV_TEST_DEPLOY_OPERATE.md` – Workflow from dev → deploy → operate.
- `RUNBOOKS.md` – Incident response playbooks for frontend issues.

These docs will be informed by the existing build/test config (Vite, Jest/Vitest, etc.) and deployment pipelines.
