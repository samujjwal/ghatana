# YAPPC App-Creator Web – User Manual

## 1. Who This Is For

This manual is for users of the app-creator UI.

## 2. Core Flows

- Open the app-creator and start a new application.
- Use the canvas and components to design pages.
- Save and later load existing applications.

Depending on your environment, you may also publish applications or export definitions through the backend integration.

This manual is self-contained and describes basic usage for the app-creator web app.
