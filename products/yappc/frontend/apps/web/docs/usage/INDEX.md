# YAPPC App-Creator Web – Usage, Reference & Backlog

This directory groups **usage-oriented** docs for the YAPPC app-creator web app.

## Planned documents

- `TECHNICAL_REFERENCE.md` – Routes, key components, and integration points.
- `USER_MANUAL.md` – How users build apps using the app-creator.
- `KNOWN_ISSUES_TROUBLESHOOTING.md` – Known UI issues and troubleshooting.
- `FUTURE_BACKLOG.md` – App-creator-specific backlog and future enhancements.

These will help onboard new users and contributors to the app-creator frontend.
