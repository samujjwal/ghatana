# YAPPC Frontend Scripts

Workspace scripts for validation, seeding, analysis, and developer automation in the YAPPC frontend stack.

## Scope
- Build, verification, and governance scripts.
- Seed and utility scripts for local development workflows.
- Frontend-specific automation that does not belong in shared platform tooling.

## Key Areas
- Validation and architecture checks.
- Local seeding and maintenance helpers.
- Script-level test coverage where present.

## Audit Notes
- Keep script behavior deterministic and safe for local execution.
- Prefer focused automation over broad scripts with hidden side effects.
