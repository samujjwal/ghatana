# YAPPC Frontend Shortcuts Library

Keyboard shortcut definitions, helpers, and integration points for YAPPC frontend applications.

## Scope
- Shared shortcut registration and metadata.
- User-facing command bindings reused across YAPPC UI surfaces.
- Supporting types and helpers for shortcut-aware components.

## Key Areas
- Shortcut constants and registries.
- Shared hook or utility integration.
- Product-local command metadata.

## Audit Notes
- Keep shortcut definitions centralized and consistent across surfaces.
- Avoid duplicating keybinding logic in individual components.
