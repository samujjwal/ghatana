# YAPPC Frontend A11y Library

Accessibility-focused utilities and components used by the YAPPC frontend workspace.

## Scope
- Accessibility helpers, behavior hooks, and test-facing utilities.
- Reusable frontend logic that improves keyboard, focus, and assistive support.
- Product-local accessibility patterns that are not yet promoted to shared platform packages.

## Key Areas
- Accessibility utilities.
- Shared helpers for accessible UI behavior.
- Supporting tests or fixtures.

## Audit Notes
- Prefer semantic markup and explicit behavior over one-off patches.
- Keep accessibility behavior reusable and easy to verify in tests.
