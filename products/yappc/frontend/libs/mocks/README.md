# YAPPC Frontend Mocks Library

Reusable test doubles, fixtures, and mock helpers for YAPPC frontend packages.

## Scope
- Mock implementations for frontend services and API clients.
- Shared fixtures used in tests, stories, and local development flows.
- Product-local testing helpers that reduce repeated setup.

## Key Areas
- Mock handlers and static fixtures.
- Test-only helper utilities.
- Shared fake data setup for frontend packages.

## Audit Notes
- Keep mocks aligned with current contracts and payload shapes.
- Prefer deterministic fixtures over highly implicit test setup.
