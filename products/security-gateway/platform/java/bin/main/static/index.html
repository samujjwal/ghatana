API Key Authentication Example

This example demonstrates how to use API key authentication to protect API endpoints.

Available Endpoints

Public Endpoint

This endpoint is public and does not require authentication.

GET /public

Protected Endpoint

This endpoint requires API key authentication.

GET /protected

Admin Endpoint

This endpoint requires API key authentication with the ADMIN role.

GET /admin

API Key Management

This endpoint allows you to list and create API keys.

GET /api-keys

POST /api-keys?name=NAME&description=DESCRIPTION&owner=OWNER
