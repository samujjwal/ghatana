1. Backend `/capabilities` endpoint exists and is authoritative.
2. All route manifest capability keys exist in backend contract.
3. No `Coming Soon`, placeholder, deterministic stub, fake AI, fake report, or fake analytics route is visible in production.
4. Google Ads command/workflow path is durable and tested.
5. Real Kernel authz/audit/approval/compliance bridge is wired.
6. Event sourcing uses typed events and aggregate-specific replay.
7. AI model governance has eval + approval + audit before promotion.
8. Real E2E tests run against backend + PostgreSQL.
9. README/audit/OpenAPI/route manifest are in sync.
10. Security tests prove forged headers cannot escalate tenant, workspace, role, or permissions.
