# Data Cloud Product Docs

This folder owns customer and product-facing Data Cloud positioning.

Use the canonical language:

- Planes are the product architecture.
- Surfaces are user and API entry points.
- Modules are implementation units.
- Runtime truth is the live, degraded, disabled, or unavailable state of planes and surfaces.
- AEP is the Action Plane runtime implementation.
