# Virtual-Org – Service – User Manual

## 1. Audience

This manual is for engineers and operators who consume or maintain `virtual-org-service`.

## 2. Basic Usage

- Upstream clients call APIs to:
  - Create and manage organizations and departments.
  - Configure and observe agents and workflows.
  - Retrieve KPIs and organization events.

## 3. Best Practices

- Use idempotent operations where possible for automation.
- Respect tenant and organization boundaries in all calls.

This manual is self-contained and explains how to use the `virtual-org-service` application in typical flows.
