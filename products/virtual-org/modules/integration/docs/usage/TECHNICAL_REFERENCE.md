# Virtual-Org – Integration Library – Technical Reference

## 1. Overview

This reference summarizes the main technical surfaces of the Virtual‑Org Integration library.

## 2. Core Concepts (Conceptual)

- **Integration helpers** that connect Virtual‑Org to:
  - Operator framework types and operators.
  - AEP pattern and learning components.
  - Shared domain models.
- **Mapping and coordination functions** that ensure consistent IDs, timestamps, and schemas across boundaries.

## 3. Intended Consumers

- Services that coordinate Virtual‑Org behavior with the operator framework and AEP.

This technical reference is self-contained and describes the primary surfaces of the Virtual‑Org Integration library.
