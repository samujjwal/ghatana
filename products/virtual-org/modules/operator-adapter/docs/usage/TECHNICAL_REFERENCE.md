# Virtual-Org – Core Operator Adapter – Technical Reference

## 1. Overview

This reference summarizes key technical concepts and surfaces of the core Operator Adapter library.

## 2. Core Concepts (Conceptual)

- **Operator adapters** that connect Virtual‑Org agents to operator framework interfaces.
- **Serialization and contract helpers** for protobuf and domain models.
- **Observability hooks** that expose metrics and traces for adapter flows.

## 3. Intended Consumers

- Platform services that embed Virtual‑Org agents into the unified operator framework.

This technical reference is self-contained and describes the primary surfaces of the core Operator Adapter library.
