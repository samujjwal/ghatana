# Virtual-Org – Java VirtualOrg-Agent Module – Known Issues & Troubleshooting

## 1. Known Issues

- **Divergent agent behaviors across services** – Implementations may vary, leading to inconsistent behavior.

## 2. Troubleshooting Scenarios

### 2.1 Inconsistent Agent Behavior

- **Symptom**: Agents of the same role behave differently across services.
- **Checks**:
  - Verify all services use the same module version.
  - Compare agent implementations and configuration.

This document is self-contained and lists common issues and mitigations for the Java `virtualorg-agent` module.
