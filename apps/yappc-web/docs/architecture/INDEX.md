# YAPPC App-Creator Web – Architecture Docs

This directory groups **design & architecture** docs for the YAPPC app-creator web app.

## Core source

- Product roadmap & architecture docs live under:
  - `products/yappc/docs/`

App-specific implementation plans (e.g., for page-builder and canvas) may also live under `docs/products/yappc/app-creator/...`.

## Planned sub-documents

- `FRONTEND_ARCHITECTURE.md` – Routing, layout, major feature modules.
- `CANVAS_AND_STATE_ARCHITECTURE.md` – Canvas design and StateManager integration.
- `DESIGN_SYSTEM_INTEGRATION.md` – Usage of `@ghatana/ui` and shared tokens.
- `INTEGRATIONS_AND_DEPENDENCIES.md` – How this app integrates with YAPPC backend services.

These docs will be aligned with the real code in this app and its supporting libs.
