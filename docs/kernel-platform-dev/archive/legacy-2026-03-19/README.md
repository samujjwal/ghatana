# Legacy Archive - 2026-03-19

**Status**: Historical archive  
**Purpose**: Preserve pre-consolidation kernel-platform-dev material that is no longer part of the active authoritative document set

---

## 1. What This Archive Is

These files were moved out of the top level during the 2026-03-19 consolidation of `docs/kernel-platform-dev`.

They are preserved because they still contain:

- raw ideas
- code examples
- migration history
- planning details
- useful terminology and implementation notes

They are not the active planning source of truth.

---

## 2. Why They Were Archived

The archived files had one or more of these problems:

- redundancy with newer convergence documents
- stale assumptions about current code reality
- overlapping plans that made the active folder hard to navigate
- overclaimed readiness or completion
- product/domain examples embedded in what should now be generic kernel guidance
