# Documentation Archive

> **Status:** Historical | **Scope:** Monorepo-wide

This directory contains historical documentation, completed audits, superseded plans, and reference material that is no longer actively maintained.

## Contents

- `audits/` - Historical audit reports
- `plans/` - Completed or superseded implementation plans
- `execution-plans/` - Historical execution plans
- `sessions/` - Session summaries and notes

## Note

Documents in this archive are preserved for reference but should not be used as current authority. For active documentation, see the parent directory or [../README.md](../README.md).

## Adding to Archive

When archiving documents:
1. Move the file to the appropriate subdirectory
2. Add a note indicating what document superseded it
3. Do not delete - preserve for historical reference
